# BC in OCaml

This is a bc evaluator in OCaml for the Programming Language Concepts class

## Usage

### Installing

This project requires:
- OCaml
- Opam
- Dune
	- `opam install dune`
- Core library
	- `opam install core`

### Running

To run the unit and integration test from the terminal:

```bash
dune runtest
```
